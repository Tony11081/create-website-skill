## Etsy Search Working Model

Optimize for the full listing, not the title alone:

- Title
- Tags
- Category
- Attributes
- Description
- Images and video
- Price, shipping, reviews, shop trust, and conversion behavior

Use structured fields precisely. Category and attributes can carry terms that do not need to be repeated as weak tags.

## Title Rules

Preferred formula:

`Specific color/material/style + product type + use case/room + buyer intent`

Examples:

- `Rust Corduroy Rocking Chair, Wood Frame Nursery Rocker`
- `Burgundy Corduroy Glider Rocking Chair, Wood Frame Nursery Rocker`
- `Wood Frame Nursery Rocking Chair, Upholstered Reading Chair`

Rules:

- Put the strongest phrase in the first 40 characters.
- Lead with product type plus differentiator: color, material, style, room, or audience.
- Keep the title natural and readable.
- Avoid repeated words.
- Avoid empty modifiers: `High-Quality`, `Perfect`, `Amazing`, `Best`.
- Avoid promo terms: `Sale`, `Free Shipping`, `Discount`.
- Avoid brand/IP/celebrity/designer terms unless licensed or factual.

Score title/SEO on:

- Query relevance and long-tail coverage.
- First 40-character strength.
- Accuracy of material, mechanism, and use case.
- Readability and buyer clarity.
- Absence of stuffing or risky claims.

## Tags Rules

Default format for this user: exactly 13 tags, one line, English commas.

Use multi-word long-tail phrases. Avoid duplicate variations that add no intent.

Good furniture tag angles:

- Product: `rocking chair`, `glider chair`, `nursery rocker`
- Material/texture: `corduroy chair`, `upholstered chair`, `wood frame chair`
- Color: `rust orange chair`, `burgundy rocker`, `charcoal chair`
- Room/use: `reading nook chair`, `bedroom chair`, `living room chair`
- Style: `mid century chair`, `accent rocker`
- Audience/occasion: `new mom gift`, `baby shower gift`

Default rocking chair tag set pattern:

`[color] chair, corduroy chair, rocking chair, nursery rocker, glider chair, wood frame chair, reading nook chair, accent rocker, mid century chair, upholstered chair, bedroom chair, living room chair, baby shower gift`

If `glider` is uncertain, replace `glider chair` with `nursery chair`, `rocker chair`, or another accurate phrase.

## Description Rules

The first three lines must answer:

- What is it?
- What makes it desirable?
- Where or why will the buyer use it?

Opening formula:

`Add [feeling/benefit] to your [room/use case] with this [specific product].`
`The [material/texture/color] and [frame/style/detail] create [style/emotional outcome].`
`Perfect for [use case 1], [use case 2], or [buyer/occasion].`

Full description sections:

- `Why You'll Love It`
- `Product Details`
- `Perfect For`
- `Care Instructions`
- `Shipping & Processing`
- `Important Notes`

Never invent dimensions, weight capacity, production time, shipping time, assembly steps, or material facts. Use placeholders.

## Visual CRO Rules

First image:

- Use a clean lifestyle scene when it improves click-through, but keep the product dominant.
- Product should occupy roughly 70-80% of the image for furniture.
- Avoid text overlays, borders, clutter, and confusing props.
- Keep color accurate and consistent across images.

Recommended image set for furniture:

- Lifestyle hero image.
- Clean product-only angle.
- Side profile showing frame/base.
- Back view.
- Fabric/material close-up.
- Scale image with person or room reference.
- Dimension diagram.
- Packaging/assembly image.
- Short video showing rocking/gliding motion and stability.

## Conversion Strategy

For premium furniture, position around use case and design identity:

- `premium nursery rocker`
- `cozy reading chair`
- `mid century accent chair`
- `statement chair for warm modern interiors`

Ads should start with long-tail intent terms, not broad single terms:

- `nursery rocker`
- `corduroy chair`
- `wood frame rocking chair`
- `reading nook chair`
- `rust orange chair`

Bundle ideas:

- Chair + matching pillow.
- Chair + ottoman.
- Chair + custom cushion cover.
- Personalized name/signature add-on.
